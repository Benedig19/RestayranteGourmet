# 🍽️ Gourmet Express

Sitio web de pedidos online para restaurante gourmet con temática peruana del Cusco.

## 🚀 Demo en vivo
[Ver página web](https://TU-USUARIO.github.io/gourmet-express)

## 📁 Estructura
- `index.html` — Página principal (menú, carrito, checkout)
- `login.html` — Sistema de login (frontend)
- `styles.css` — Estilos CSS completos
- `app.js` — Lógica del carrito, checkout y productos

## ✨ Características
- Catálogo de 60 productos auténticos del Cusco, Perú
- Carrito de compras funcional
- Checkout de 4 pasos
- Tema oscuro / claro
- Diseño responsive
- Animaciones profesionales

## 🛠️ Tecnologías
HTML5, CSS3, JavaScript vanilla
